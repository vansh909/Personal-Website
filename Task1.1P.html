Library Book Sorting

The smart parking system implemented in the college campus utilizes RFID sensors for identification, IR sensors for slot detection, and DHT sensors for temperature management inside indoor parking slots. The system efficiently manages parking by differentiating between authentic users and normal users, providing them with designated parking slots. Real-time data from RFID and IR sensors allows for accurate information on slot availability to be displayed on screens, enhancing user experience. Additionally, the DHT sensor maintains optimal temperature inside indoor parking spaces and automatically activates exhaust fans when needed. The project aims to improve parking efficiency, user convenience, and space utilization, offering a scalable and future-ready solution for modern parking challenges.

The "Library Book Sorter" project is an autonomous robotic system designed to streamline the book sorting process in libraries. The robot is equipped with obstacle detection sensors, allowing it to navigate safely and efficiently through the library space. When the robot detects a book, it halts, and a robotic arm picks up the book. An ESP32 microcontroller scans the book's QR code, determining its designated shelf location. Utilizing this information, the robot car navigates to the correct shelf and, with the assistance of the robotic arm, places the book in its appropriate spot
